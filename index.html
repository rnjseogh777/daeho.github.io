FAQ

어떤 용도로 쓸 수 있나요?

개인 포트폴리오, 소규모 비즈니스 랜딩, 프로젝트 소개용 등으로 바로 사용 가능합니다. 섹션 텍스트만 바꾸세요.

배포는 어떻게 하나요?

GitHub Pages, Netlify, Vercel 중 하나에 index.html을 업로드하면 됩니다. 이미 안내한 순서를 따르면 10분 내 배포 가능.

문의 폼은 작동하나요?

기본은 안내 알림만 띄웁니다. 실제 수집은 Formspree, Google Apps Script 등과 연동해 주세요.

디자인은 변경할 수 있나요?

가능합니다. 색상 변수, 섹션 순서, 카드 개수 등을 자유롭게 수정하세요.
